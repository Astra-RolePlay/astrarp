# Features
- Custom UI
- Configurable
- Optimized (0.0ms idle)

# Installation
1. Drag `Ricky-VinewoodSign` in your resource folder
2. Setup config
3. Add `ensure Ricky-VinewoodSign` in your server.cfg
4. Start the server.
